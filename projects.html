Wireshark Malware Traffic Analysis

I analyzed a PCAP file using Wireshark to uncover malware communications. Suspicious connections to malicious domains were identified. The investigation revealed a Trojan/backdoor infection. This project reinforced my ability to detect and trace malicious network traffic.

PowerShell Keylogger Analysis

I examined a malicious PowerShell script disguised as a keylogger. The script exfiltrated keystrokes via SMTP to an attacker’s email. Hardcoded credentials and DLL imports were uncovered during review. This project highlighted how attackers abuse PowerShell for persistence and data theft.

SQL Injection Attack

In this project, I exploited a vulnerable login form using SQL injection. By manipulating query logic, I bypassed authentication and accessed restricted data. The test demonstrated the dangers of unsanitized inputs and poor query handling. This reinforced the importance of parameterized queries in defensive coding.

Shopping Basket Exploit

I intercepted HTTP requests to gain access to another user’s shopping basket. Using Burp Suite, I manipulated session tokens and bypassed access controls. This allowed me to view and alter another user’s private shopping data. The exercise highlighted broken access control as a critical OWASP vulnerability.

Zero-Star Review Injection

For this challenge, I modified HTTP requests to submit a zero-star product review. The frontend blocked this input, but intercepting traffic allowed the request to bypass validation. This exposed a lack of server-side enforcement for input constraints. It reinforced the need for backend validation alongside frontend controls.

Brute Force Admin Login

I used Burp Suite with a password wordlist to brute force the admin login. Once credentials were discovered, I reset the admin password to demonstrate full account takeover. This scenario revealed how weak password policies increase attack surface. It underscored the need for account lockouts, MFA, and monitoring to stop brute force attempts.

Hidden JSON File Access

I discovered and accessed hidden documents within the Juice Shop environment. By using a null byte injection, I was able to retrieve sensitive JSON files. This revealed improper file access controls and directory traversal flaws. The project emphasized secure file path handling and validation as essential defenses.
