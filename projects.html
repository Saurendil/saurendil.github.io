PowerShell Script Analysis

Investigated a malicious PowerShell keylogger script discovered in a compromised system. Found hardcoded credentials, SMTP-based exfiltration, and DLL imports used for persistence. Reverse engineered script logic to uncover its capabilities. Documented indicators of compromise (IOCs) including email accounts, registry changes, and process creation. Provided remediation steps to remove persistence and block exfiltration channels.

Offensive Projects

OWASP Juice Shop Exploitation

Conducted penetration testing challenges against the OWASP Juice Shop platform. Exploited multiple vulnerabilities including SQL injection, cross-site scripting (XSS), CAPTCHA bypass, broken access control, weak password authentication, forgotten backups, and insecure product review flows. For each exploit, I documented proof-of-concept requests and explained developer-focused remediation steps. These challenges improved my ability to identify, exploit, and explain issues aligned to the OWASP Top 10. This work deepened my understanding of web application vulnerabilities and secure coding practices.
