Malware Deep Investigation

Investigated a malicious PowerShell keylogger script designed to capture keystrokes and exfiltrate them via SMTP. Discovered hardcoded email credentials, DLL imports, and persistence mechanisms such as registry modification. Broke down obfuscation, tracked file write operations, and identified clear indicators of compromise (IOCs). Documented how the script communicated with external mail servers and recommended mitigations including blocking outbound SMTP traffic, strengthening endpoint monitoring, and enforcing PowerShell execution policies. Reinforced my malware reverse engineering and detection skills in a defensive context.

Offensive Projects

OWASP Juice Shop Exploitation

Completed penetration testing challenges in the OWASP Juice Shop application, targeting vulnerabilities across the OWASP Top 10. Successfully exploited SQL injection to bypass login authentication, performed stored XSS to trigger malicious scripts, and bypassed CAPTCHA mechanisms using automation. Identified weak credentials in authentication flows and retrieved forgotten backup files via path traversal. Discovered insecure review and rating systems that allowed manipulation of product reputations. For each exploit, created proof-of-concept payloads and recommended developer mitigations such as parameterized queries, stronger session management, and improved access controls. These exercises reinforced offensive testing methodology and improved my ability to map vulnerabilities to their corresponding mitigations.
