Find the Culprit – Malware Traffic Analysis

In this project, I investigated a PCAP file to uncover signs of compromise. Using Wireshark, I tracked suspicious DNS and HTTP traffic tied to malware activity. The analysis revealed a Windows host communicating with malicious domains. I confirmed the infection type as a Trojan/backdoor targeting system persistence.

Malicious Email Analysis

This project focused on dissecting a phishing email with a suspicious attachment. I examined email headers, sender IPs, and embedded links to trace potential spoofing. The attachment was flagged as a delivery method for credential theft. The findings reinforced detection techniques for social engineering attacks.

PowerShell Script Analysis

I reverse engineered a malicious PowerShell script designed as a keylogger. The script exfiltrated keystrokes via SMTP to an attacker’s email account. It contained hardcoded credentials and DLL imports for stealthy persistence. This exercise demonstrated the importance of script auditing in endpoint security.

Linux Log Review

For this project, I analyzed Linux log files to identify unusual system behavior. Suspicious login attempts, privilege escalation errors, and cron job entries were discovered. The timeline of events suggested a brute force attempt followed by lateral movement. This reinforced my skills in log analysis for incident detection and response.
